# Ticket System API

REST API for ticket management built with ASP.NET Core.

## Tech Stack

- C#
- ASP.NET Core
- Entity Framework
- SQL Server / PostgreSQL
- JWT Authentication

## Features

- User authentication
- Create tickets
- Update ticket status
- Assign tickets to users
- RESTful architecture

## Installation

1. Clone the repository
2. Configure database connection
3. Run migrations
4. Start the API
